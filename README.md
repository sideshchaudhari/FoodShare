# FoodShare
FoodShare – Smart Food Donation Web App

About the Project

FoodShare is a web-based platform designed to reduce food wastage by connecting food donors with nearby NGOs and people in need. It provides a simple and efficient way for users to donate excess food, ensuring it reaches the right place at the right time.

Key Features
Easy food donation system for donors
Location-based matching with nearby NGOs
Smart matching using a basic KNN model
Real-time notifications for requests
Multiple user roles: Donor, NGO, Admin
Simple and user-friendly dashboard

Objective:

The main objective of FoodShare is to minimize food waste and create a system where surplus food can be distributed efficiently to those who need it the most.

Tech Stack:

Frontend is built using HTML,CSS,JS,Bootstrap.
Backend is developed using Flask.
Database is MySQL.
A basic KNN algorithm is used for smart matching.

Project Structure

The project is divided into frontend, backend, machine learning model, and database configuration sections. Each part works together to provide a complete working system.

Setup Instructions

Clone the repository from GitHub and install the required dependencies for both frontend and backend.
Run the backend server first, then start the frontend application to access the system locally.

Implementation Cost

This project is developed as a low-cost student solution. The main cost is in frontend and backend development, while other components like database and hosting use free or minimal-cost services. The overall implementation cost is approximately ₹15,000 to ₹25,000.

Future Enhancements

The project can be improved by adding a mobile app version, real-time delivery tracking, enhanced security features, and expanding the system to multiple cities.
